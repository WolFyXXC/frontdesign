# Лабораторная работа по JavaScript

## 📋 Описание проекта
Веб-приложение на JavaScript, демонстрирующее базовые алгоритмы и работу с DOM.

## 🚀 Функциональность
- **Калькулятор суммы** - сложение двух чисел
- **Проверка простых чисел** - определение, является ли число простым
- **Генератор случайных чисел** - генерация числа от 1 до 10000 и вывод всех его делителей

## 🛠 Технологии
- HTML5
- CSS3 (Flexbox, Grid, Анимации)
- JavaScript (ES6+)

## 📁 Структура проекта

📦 project

├── 📁 js

## 🎨 Особенности дизайна
- Градиентный фон с анимацией
- Адаптивный дизайн для мобильных устройств
- Плавные анимации при взаимодействии
- Кастомный скроллбар
- Стеклянный эффект (glassmorphism) для контейнера

## 🔧 Установка и запуск
1. Скачайте все файлы проекта
2. Откройте `index.html` в любом современном браузере
3. Готово! 🎉

## 📝 Примечания
- Все числа генерируются и обрабатываются на стороне клиента
- При обновлении страницы генерируется новое случайное число
- Интерфейс полностью адаптирован под мобильные устройства
